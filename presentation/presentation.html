🔄 Процес обробки запиту

sequenceDiagram participant User as 👤 Користувач participant Browser as Веб-браузер participant Flask as Flask App participant Agent as Tarot Agent participant Selector as Card Selector participant VectorDB as ChromaDB participant RAG as RAG Chain participant LLM as GPT-4 API participant Monitor as Observability Note over User,Monitor: Процес отримання Таро читання User->>Browser: Вводить питання Browser->>Flask: POST /api/reading activate Flask Flask->>Agent: get_reading(question) activate Agent Agent->>Monitor: Логування початку запиту Note over Agent,Selector: Вибір карт Agent->>Selector: _draw_cards(num_cards=3) Selector-->>Agent: Список карт + положення Note over Agent,VectorDB: Пошук інформації про карти loop Для кожної карти Agent->>VectorDB: similarity_search(card_name) VectorDB-->>Agent: Описи карти (top-k=5) end Note over Agent,RAG: Формування контексту Agent->>RAG: Створення контексту з описів RAG->>Agent: Підготовлений prompt Note over Agent,LLM: Генерація інтерпретації Agent->>LLM: Запит з контекстом LLM-->>Agent: Інтерпретація українською Note over Agent,Monitor: Обробка результату Agent->>Monitor: Логування успішної відповіді Agent->>Agent: Формування відповіді Agent-->>Flask: {"cards": [...], "reading": "..."} deactivate Agent Flask-->>Browser: JSON відповідь deactivate Flask Browser-->>User: Відображення карт та інтерпретації
