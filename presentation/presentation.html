🎯 Огляд проекту

Tarot AI Agent - це інноваційна система для автоматизації оцінки ризиків та підтримки прийняття рішень через поєднання традиційних методик з сучасними AI технологіями.

🤖 Штучний інтелект

GPT-4 Turbo для генерації інтерпретацій та RAG система для контекстного пошуку

📊 Векторна база

ChromaDB для швидкого пошуку релевантної інформації серед 1000+ документів

🌐 Веб-інтерфейс

Зручний адаптивний UI для взаємодії з системою на будь-яких пристроях

🛡️ Система Guardrails

Комплексна безпека та контроль якості відповідей

🎯 Цілі та задачі

🚀 Автоматизація

Швидкий аналіз бізнес-ситуацій та структуровані рекомендації

💰 Оптимізація

Зниження вартості консультацій та масштабованість рішення

🔍 RAG System - Ядро системи

sequenceDiagram participant U as Користувач participant A as AI Agent participant V as Vector DB participant L as GPT-4 U->>A: Питання про ризики A->>A: Витягує випадкові карти A->>V: Шукає описи карт V-->>A: Релевантні документи A->>L: Промпт з контекстом L-->>A: Інтерпретація A-->>U: Карти + читання

📚 Retrieval (Витягування)

Пошук по 1000+ документам з використанням similarity search та метаданих для знаходження найрелевантніших описів карт

🔗 Augmentation (Збагачення)

Контекстне збагачення промптів знайденою інформацією для формування детального контексту

✨ Generation (Генерація)

GPT-4 з температурою 0.7 для креативних, але обґрунтованих інтерпретацій українською мовою

🎯 Quality Control

Система Guardrails для перевірки якості та відповідності згенерованих відповідей

🎭 Старші Аркани

22 карти - символічні архетипи та життєві уроки. Кожна карта представляє важливий етап духовного розвитку.

  • Старші життєві теми
  • Духовні аспекти
  • Кармічні уроки

🃏 Молодші Аркани

56 карт - практичні аспекти повсякденного життя, розділені на 4 масті по 14 карт.

  • Кубки - емоції, стосунки
  • Мечі - думки, конфлікти
  • Жезли - дія, творчість
  • Пентаклі - матеріальне
Технічна реалізація: SentenceTransformers all-MiniLM-L6-v2 для створення семантичних ембедінгів, ChromaDB для векторного пошуку, підтримка метаданих для точного фільтрування.
